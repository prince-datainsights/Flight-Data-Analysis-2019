# Flight-Data-Analysis-2019
Project Overview:-
This project analyzes a comprehensive dataset of 10,683 domestic flights in India from (2019), covering key attributes
such as airline, date of journey, source, destination, route, departure and arrival times, duration, total stops, additional information, and ticket prices.

Objectives:-
The objective of this analysis is to extract actionable insights related to flight distributions, pricing trends, and travel patterns.
The project explores various analytical questions, including:
* Distribution of flights by number of stops
* Identification of the most expensive and cheapest flight categories
* Price variations across source cities
* Flight frequency by day and month
* Impact of additional services on ticket pricing.

Dataset Information
* number of rows: 10799
* number of columns: 11
* File Used: Data_Train_xlsx
Type: Indian Domestic Flights

Key Columns:
* Airline-Name of the airline operating the flight.
* Source- Departure city
* Destination- Arrival city
* Duration- Total travel time of the flight
* Total_Stops-Number of stops between source and destination
* Additional_Info- Extra service details
* Price- Ticket price (in INR)


 Tools I used 
* Python-For data cleaning and analysis
* Pandas- For structure data manipulation and analysis.
* Matplotlib- (Data visualization)
* Seaborn- (Statistical visualization)
* Jupyter Notebook - To write ,run and visualize the code
